# ShopEase 🛒
Sri Lanka's Full-Stack E-Commerce Management System built with Next.js + MongoDB.

## Features

- 5-page public website: Home, About, Products, Cart, Contact
- Separate admin panel with secure login dashboard
- Contact form with full backend integration & MongoDB storage
- JWT-based admin authentication
- Responsive UI with Tailwind CSS
- Contact message management (view & delete) in admin panel
- Form validation with success/error feedback
- Clean, reusable component architecture

## Pages

| Route | Description |
|---|---|
| Home (`/`) | Landing page with featured products and highlights |
| About (`/about`) | Brand story and team information |
| Products (`/products`) | Full product listing |
| Cart (`/cart`) | Shopping cart and order summary |
| Contact (`/contact`) | Contact form with backend submission |
| Admin Login (`/admin/login`) | Secure admin authentication |
| Admin Dashboard (`/admin/dashboard`) | Contact submissions management |

## Getting Started

```bash
# Install all dependencies
npm install

# Run both apps concurrently
npm run dev
```

| App | URL |
|---|---|
| Website | http://localhost:3000 |
| Admin Panel | http://localhost:3001 |

## Tech Stack

- **Frontend:** Next.js 15 (App Router)
- **Backend:** Next.js API Routes
- **Database:** MongoDB + Mongoose
- **Styling:** Tailwind CSS
- **Authentication:** JWT / Session Authentication
- **Language:** TypeScript
